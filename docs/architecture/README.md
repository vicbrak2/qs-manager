# Architecture Docs

Espacio para diagramas, decisiones de estructura y guias tecnicas del plugin `qs-core`.

## Referencias activas

- [ADR-011 Boilerplate generico](../adr/ADR-011-generic-wp-boilerplate.md): documenta la decision de extraer el nucleo reutilizable de `qs-manager` hacia un template independiente.
- [Plan del boilerplate](../generic-wp-boilerplate-plan.md): roadmap de auditoria, extraccion, tooling CLI y documentacion.

## Estado actual

El repositorio actual sigue siendo el producto `qs-core` desplegable para WordPress. La extraccion a boilerplate es una iniciativa posterior y no reemplaza la estructura ni el flujo de deploy vigentes en este repo.
