Sistema interno de gestion para QamilunaStudio.

== Description ==

QS Core es un plugin monolitico modular para operacion interna, integracion con LatePoint y gestion administrativa.

== Installation ==

1. Instala dependencias con `composer install`.
2. Copia el directorio completo a `wp-content/plugins/qs-core`.
3. Activa el plugin desde WordPress usando `qs-core.php`.
