# QS Manager

Sistema de gestión orientado a un estudio de maquillaje.

## Módulos Previstos
*   **Agenda y Citas**: Calendario de servicios, horas disponibles, recordatorios.
*   **Gestión de Clientes**: Historial de clientes, preferencias y datos de contacto.
*   **Inventario**: Control de productos y materiales.
*   **Finanzas Básicas**: Registro de cobros, control de gastos, rentabilidad.
